**HelloApp** is a lightweight, console-based application designed to demonstrate fundamental programming concepts, basic input/output operations, and environment configuration. It serves as a foundational stepping stone for setting up a clean development workflow.

## ✨ Features Demonstrated

* **Basic I/O:** Capturing user input from the console and rendering formatted output.
* **Clean Code Structure:** Simple, readable, and well-commented source code illustrating best practices for introductory software design.
* **Environment Verification:** Acts as a reliable test project to verify that the compiler and local development environment are set up correctly.
